NOTICE D’UTILISATION

 

L’application permet de lire et de modifier des images PPM de type P1, P2, P3 et P6.

L’enregistrement d’une image se fait toujours au format P3.


L’application se divise en trois fenêtres :


  1. La fenêtre principale qui permet d’effectuer toutes les actions sur les images de type Portable PixMap.


  1. La fenêtre affichant l’image originale.


  1. La fenêtre affichant l’image modifiée.

3.    Ouverture et sauvegarde

Pour ouvrir une image :

Fichier → Ouvrir puis sélectionner l’image à ouvrir.


Pour sauvegarder une image :

Fichier → Enregistrer.


Pour sauvegarder une image sous un autre répertoire/nom :

Fichier → Enregistrer sous puis sélectionner le répertoire et le nouveau nom de l’image.


Remarque :

Lors de l’ouverture ou de la sauvegarde, les fichiers doivent toujours avoir l’extension ‘.ppm’
